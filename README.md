# Dugout Online API Wrapper

Ever wanted to write a tool for Dugout Online game, but it was hard to even start
due to lack of a public API? This is the unofficial API you were looking for!

## Features

### Move players to youth team and back
- DO.Squad.movePlayerToFirstTeam(playerId)
- DO.Squad.movePlayerToYouthTeam(playerId)
- DO.Squad.movePlayersToFirstTeam(playerIds)
- DO.Squad.movePlayersToYouthTeam(playerIds)
